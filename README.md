### Installation T-Remix
* `Commands` for termux
```
$ termux-setup-storage
  
$ pkg update && pkg upgrade && pkg install git -y

$ git clone https://github.com/Arij-arman/T-Remix

$ ls

$ cd T-Remix

$ ls

$ chmod +x *

$ bash t-remix.sh
```



### To Remove Banner
* `Commands` for termux
```
$ cd T-Remix

$ ls

$ bash remove-banner.sh
```
### To Remove Login
* `Commands` for termux
```
$ cd T-Remix

$ ls

$ bash remove-login.sh
```
